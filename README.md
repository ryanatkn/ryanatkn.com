## usage

```bash
# node >= 16.6
npm i
# then
npm run dev
# or
gro dev # npm i -g @feltjs/gro
```

[Vite](https://github.com/vitejs/vite),
and [Gro](https://github.com/feltjs/gro) for more.

```bash
npm run build
# or
gro build
```
